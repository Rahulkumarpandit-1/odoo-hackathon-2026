# FleetFlow Frontend

Frontend application for **FleetFlow™ – Enterprise Fleet Management System**.

Built using **React + Vite**, this client application provides a responsive dashboard for managing vehicles, drivers, trips, maintenance logs, and expenses.

---

## Tech Stack

- React.js
- Vite
- JavaScript (ES6+)
- REST API Integration
- JWT Authentication

---

## Features

- Secure Login System
- Role-based Dashboard
- Vehicle Registry
- Driver Management
- Trip Dispatching
- Maintenance Logs
- Expense Tracking
- Analytics Page

---
